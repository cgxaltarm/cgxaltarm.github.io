DAILYFLOW HYBRID TOPOLOGY DASHBOARD

Tampilan:
- Gaya interaktif DailyFlow lama
- 9Router sebagai node pusat
- Provider mengelilingi node utama
- Garis routing bergerak
- Status aktif, lambat, offline, idle
- Klik node untuk detail token, request, latency, model, error
- Aktivitas terbaru
- Zoom, geser, reset
- Responsif
- Demo otomatis jika /public-usage belum aktif
- Data sensitif tidak ditampilkan

Cara pasang:
1. Ekstrak ZIP.
2. Upload index.html ke repository cgxaltarm.github.io.
3. Ganti index.html lama.
4. Commit ke branch main.
5. Tunggu GitHub Pages deploy.
6. Buka https://dailyflo.me/?v=hybrid1

Cara pasang:
1. Buka repository GitHub Pages cgxaltarm.github.io.
2. Ganti index.html lama dengan index.html dari paket ini.
3. Commit ke branch main.
4. Tunggu GitHub Pages selesai deploy.
5. Buka https://dailyflo.me.

Sumber data live:
https://router.dailyflo.me/public-usage

Jika endpoint belum tersedia, halaman otomatis memakai mode demo lokal.

Data sensitif yang tidak boleh dikirim oleh endpoint:
- apiKey
- accessToken
- refreshToken
- authorization
- secret
- password
- credential
- cookie
- isi prompt
- isi respons

Endpoint API 9Router tetap:
https://router.dailyflo.me/v1
